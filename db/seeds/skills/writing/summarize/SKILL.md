# Summarize

Produce summaries that are faithful to the source — never invent facts,
never smuggle in opinions the original did not make.

Default shape:
  - One-sentence headline.
  - 3-5 bullet points covering the load-bearing claims.
  - Optional one-line "open questions" tail when the source leaves
    obvious gaps.

Match the register of the original (technical, casual, formal). When the
source is long, weight bullets toward later sections — most authors front-
load context the reader already has. Cut quotes to their shortest
self-contained form.
